## 1. 프로젝트 개요

### 1.1 목적
해외여행 시 항공 수하물 규정을 쉽게 확인하고, 여행 경험을 공유할 수 있는 모바일 앱

### 1.2 배경
- 항공사마다 수하물 규정이 달라 여행자가 헷갈리는 경우가 많음
- 카메라로 물품을 찍으면 반입 가능 여부를 즉시 확인할 수 있으면 편리함
- 같은 나라를 여행한 사람들의 후기를 공유하는 커뮤니티 필요

### 1.3 대상 사용자
- 해외여행을 준비하는 내국인
- 수하물 규정이 헷갈리는 여행자
- 여행 후기를 공유하고 싶은 사용자

## 2. 기능 요구사항

|----|------|------|----------|

## 3. 비기능 요구사항

### 3.1 보안
- 비밀번호는 BCrypt로 암호화 저장
- JWT 토큰 만료 시간 1시간
- DB 접속 정보 및 API 키는 환경변수로 관리 (코드에 하드코딩 금지)
- CORS 설정으로 허용된 출처만 접근 가능

### 3.2 성능
- API 응답 시간 2초 이내 (AI 분석 제외)
- AI 분석 응답 시간 5초 이내

### 3.3 운영
- 무료 플랜으로 운영 가능한 구조
- 서버 다운 시 에러 메시지 표시

### 3.4 사용성
- 로딩 시 ActivityIndicator 표시
- 에러 발생 시 사용자 친화적 메시지 표시
- iOS / Android 동시 지원
